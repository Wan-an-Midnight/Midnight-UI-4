About Midnight UI 4

Midnight UI 4.0 will be fully launched with a new logo to be announced on 21 July 2022 and a new website name Midnight UI1。
Designed and developed by the CEO and led by the CTO for six months, Midnight UI 4 is a complete upgrade from the previous version, using the new Bootstrap 5.21 framework and the MNUIwebKit 1.0 framework developed independently by Midnight, bringing new features and a different experience than ever before.

To find out more about Midningt's website updates, visit: https://www.midnightnth.club/page/updates/HT201222
