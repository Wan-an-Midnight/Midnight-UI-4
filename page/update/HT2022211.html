关于 Midnight UI 4

Midnight UI 4.0 将全面启用于2022年7月21日公布的全新 LOGO 以及新的官网名称 Midnight UI1。
经过由 CEO 亲手设计，CTO 领导下长达半年的设计与开发，Midnight UI 4 带来了相比较于上一版本的全面升级，使用全新的 Bootstrap 5.21 框架，由 Midnight 独立开发的 MNUIwebKit 1.0 框架，带来前所未有全新功能与不同凡响的体验。

了解更多 Midningt 网站更新的内容，请访问: https://www.midnightnth.club/page/updates/HT201222
